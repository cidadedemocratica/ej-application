Preparando o ambiente ¶

Python¶

O EJ requer o Python 3.6+ e um ambiente POSIX. Se você usar uma versão recente do uma distribuição Linux, ele provavelmente já está instalado em seu sistema, caso contrário você terá que seguir as instruções específicas do sistema:

  • Ubuntu <https://websiteforstudents.com/installing-the-latest-python-3-7-on-ubuntu-16-04-18-04/>

Se você tem uma distribuição relativamente recente, tente uma das instruções abaixo:

Ubuntu/Debian¶

Tudo o que você precisa é um apt de distância:

$ sudo apt update
$ sudo apt install python3-dev npm
$ sudo pip install invoke~=1.0

ArchLinux¶

Tudo que você precisa é empacotado no Pacman:

$ sudo pacman -S python npm
$ sudo pip install invoke~=1.0
